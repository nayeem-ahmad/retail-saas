# Epic 25: Purchase Settings & Configuration

### Epic Goal
Centralize all procurement-related business rules, ensuring purchases and PO workflows follow store policies.

### Epic Description
**Stories:**
1. **Story 1: Default Purchase Rules** - Configure default tax rates and cost-allocation rules (e.g., freight distribution).
2. **Story 2: Supplier Communication Templates** - Customize PO and RFQ templates for different suppliers.
3. **Story 3: Procurement Approvals** - Optional multi-level approval flow for high-value Purchase Orders.
