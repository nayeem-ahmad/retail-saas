# Epic 50: Business Expense Management

### Epic Goal
Allow store owners to record and categorize non-inventory expenses (e.g., rent, utilities, salaries) to calculate true net profit.

### Epic Description
**Stories:**
1. **Story 1: Expense Category Management** - Define custom expense groups.
2. **Story 2: Daily Expense Entry** - Simple UI for recording daily cash outflows.
3. **Story 3: Expense Reporting** - View expense breakdowns by period and category.
