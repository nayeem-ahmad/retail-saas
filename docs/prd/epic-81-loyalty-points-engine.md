# Epic 81: Loyalty Points Engine

### Epic Goal
Increase customer retention by rewarding repeat purchases through a configurable points earning and redemption system.

### Epic Description
**Stories:**
1. **Story 1: Loyalty Rule Configuration** - Define global rules (e.g., "1 point per 100 BDT spent") and redemption rates.
2. **Story 2: POS Points Integration** - Automated point calculation and display during the checkout process.
3. **Story 3: Points Redemption Logic** - Allow customers to use points as a discount on their current sale, with secure balance checks.
4. **Story 4: Points Ledger** - A transactional log of all points earned and spent by a customer.
