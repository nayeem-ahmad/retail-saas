# Epic 15: Sales Settings & Configuration

### Epic Goal
Centralize all sales-related business rules, ensuring the POS and order workflows adhere to store-specific policies.

### Epic Description
**Stories:**
1. **Story 1: Default Pricing & Tax Rules** - Configure store-wide tax percentages and default discount limits.
2. **Story 2: Invoice & Receipt Templates** - Customize headers, footers, logos, and SMS message formats.
3. **Story 3: POS Feature Toggles** - Enable/disable features like stock-reservation or customer-mandatory checkout.
