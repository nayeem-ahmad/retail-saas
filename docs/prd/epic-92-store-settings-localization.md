# Epic 92: Store-wide Settings & Localization

### Epic Goal
Centralize all store-level configurations, ensuring the platform adapts to the specific needs of different businesses and regions.

### Epic Description
**Stories:**
1. **Story 1: General Store Profile** - Manage store name, address, and contact info.
2. **Story 2: Localization Settings** - Configure currency (e.g., BDT), timezone, and date formats.
3. **Story 3: Communication Templates** - Customize SMS and receipt header/footer content.
