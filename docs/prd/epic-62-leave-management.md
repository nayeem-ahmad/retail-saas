# Epic 62: Leave Management Workflow

### Epic Goal
Automate the process of requesting, approving, and tracking staff leave to ensure operational continuity.

### Epic Description
**Stories:**
1. **Story 1: Leave Type Configuration** - Define categories like "Casual", "Sick", or "Earned" leave.
2. **Story 2: Leave Request UI** - Mobile-friendly form for staff to submit dates and reasons.
3. **Story 3: Manager Approval Workflow** - Notification and dashboard for managers to approve/reject requests.
4. **Story 4: Leave Balance Tracking** - Automatic calculation of remaining leave days per employee.
