# Epic 70: Payroll Processing

### Epic Goal
Automate the calculation and disbursement of staff salaries based on attendance, bonuses, and deductions.

### Epic Description
**Stories:**
1. **Story 1: Salary Profile Definition** - Define base pay and benefits per employee.
2. **Story 2: Monthly Payroll Generation** - Calculate net pay based on Epic 60 attendance data.
3. **Story 3: Payslip Generation** - Generate digital payslips for employees.
