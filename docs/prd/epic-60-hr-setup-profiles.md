# Epic 60: HR Setup & Employee Profiles

### Epic Goal
Establish the foundational organizational structure and maintain comprehensive digital records for all staff members.

### Epic Description
**Key Features:**
*   **Organizational Hierarchy:** Define Departments and Designations specific to the tenant.
*   **Digital Employee File:** Centralized profile for each staff member including Joining Date, NID, and contact info.
*   **System Access Control:** Link employees to system `Users` for login-enabled roles (Cashiers, Managers).

**Stories:**
1. **Story 1: Department & Designation CRUD** - Basic setup interface for the business structure.
2. **Story 2: Employee Profile Management** - Comprehensive form to add/edit staff with document upload placeholders.
3. **Story 3: Employee List View** - Filterable list of all active/inactive staff with quick-action buttons.
