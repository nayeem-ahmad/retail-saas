# Epic 30: Financial Ledgers & Core Accounting

### Epic Goal
Implement a core accounting engine that automatically records debits and credits for all store transactions (Sales, Purchases, Expenses).

### Epic Description
This is a foundational epic for the Premium Tier, enabling full financial transparency for store owners.

**Stories:**
1. **Story 1: Chart of Accounts Definition** - Define assets, liabilities, equity, revenue, and expenses.
2. **Story 2: Automated Journal Entries** - Hook into Sales and Purchase APIs to generate ledger entries.
3. **Story 3: Trial Balance & Financial Reports** - Generate basic reports (Profit & Loss, Balance Sheet).
