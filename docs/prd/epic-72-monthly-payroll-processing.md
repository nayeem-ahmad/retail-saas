# Epic 72: Monthly Payroll Processing

### Epic Goal
Automate the month-end salary calculation and generate all necessary financial records and payslips.

### Epic Description
**Stories:**
1. **Story 1: One-Click Payroll Generation** - Batch calculate salaries for all active employees based on attendance, leave, and loans.
2. **Story 2: Salary Disbursement & Voucher Sync** - Mark payroll as "Paid" and automatically generate an Accounting Voucher (Epic 30).
3. **Story 3: Digital Payslip Generation** - PDF generation and optional SMS notification of salary credit.
