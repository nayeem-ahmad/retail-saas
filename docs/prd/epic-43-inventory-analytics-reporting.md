# Epic 43: Inventory Dashboard # Epic 43: Inventory Analytics & Reporting Reporting

### Epic Goal
Provide actionable insights into inventory health, stock turnover, and automated replenishment suggestions.

### Epic Description
**Key Reports:**
*   **Stock Ledger:** Detailed transaction history for every item (In, Out, Balance).
*   **Stock-on-Hand Summary:** Current value and quantity across all warehouses.
*   **Suggest Items (Reorder List):** Intelligent report suggesting products to buy based on current levels, reorder points, and lead times.

**Stories:**
1. **Story 1: Universal Stock Ledger API** - Detailed view of every movement affecting a product's stock.
2. **Story 2: Reorder Suggestion Algorithm** - Generate a "Suggested Purchase" list based on low-stock thresholds.
3. **Story 3: Inventory Valuation Reporting** - Total value of stock using Weighted Average Cost (WAC) or FIFO.
