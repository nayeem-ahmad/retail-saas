# Epic 21: Purchase Order (PO) Management

### Epic Goal
Formalize the procurement process by allowing store managers to create, track, and receive Purchase Orders (POs) from suppliers.

### Epic Description
This epic will transition the current "simple purchase" recording into a more structured procurement workflow.

**Stories:**
1. **Story 1: PO Creation UI & API** - Draft POs with expected items and prices.
2. **Story 2: PO Approval Workflow** - Basic status tracking (Draft, Sent, Received).
3. **Story 3: PO-to-Inventory Integration** - Automatically update stock levels when a PO is marked as "Received".
