# Epic 70: Salary Profiles & Payroll Setup

### Epic Goal
Define the financial structure of employee compensation, including basic pay, allowances, and recurring deductions.

### Epic Description
**Stories:**
1. **Story 1: Salary Profile Builder** - Template-based setup for different salary grades.
2. **Story 2: Allowance & Deduction Types** - Global list of components (e.g., Medical, Mobile Bill, Tax).
3. **Story 3: Assign Profile to Employee** - Link specific financial terms to an employee profile.
